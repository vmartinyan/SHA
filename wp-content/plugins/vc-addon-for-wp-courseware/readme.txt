This plugin adds integration between Visual Composer and WP Courseware to customize unit content with Visual Composer elements.

== Description ==
[Fly Plugins](http://flyplugins.com) presents [Visual Composer](https://vc.wpbakery.com/) for [WP Courseware](http://wpcourseware.com).

= Do you want to use the power of Visual Composer and create custom course units? =
The Visual Composer integration for WP Courseware is designed to give you control of creating custom layouts for your course units.

= Disclaimer =
This is not the actual page builder plugin. It is only the integration, or extension to allow you to customize WP Courseware units with Visual Composer. Visual Composer is a separate product.

== Installation ==

1. Upload the `Visual Composer Addon for WP Courseware` folder into the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
